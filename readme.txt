# Customer Retention & Repurchase Analysis

This project is my hands-on practice in turning raw e-commerce data into a clear customer retention story.

Using the Olist dataset, I focused on a few questions that matter in real business settings:  
How many customers come back after their first purchase?  
How does retention change over time?  
And how does customer behavior differ across regions?

To answer those questions, I used SQL to prepare and analyze the data, then built a Power BI dashboard to present the results in a more business-friendly way.

## What this project covers
- Cohort retention analysis
- New vs. repeat customer trends
- 30/60/90-day repurchase analysis
- State-level customer behavior breakdown
- Interactive dashboard in Power BI

## Tools
- MySQL
- Power BI

## Dataset
- Olist Brazilian E-commerce Dataset
- Main tables used: `orders`, `customers`

## Project structure
- `data/` contains the raw dataset files used in the project
- `sql/` contains the queries for cleaning, transformation, and analysis
- `dashboard/` contains the Power BI file and dashboard screenshots

## Key takeaways
- Retention drops sharply after the first purchase month
- Most customer activity comes from new customers rather than returning ones
- Repurchase patterns become more meaningful when viewed in 30/60/90-day windows
- Customer behavior varies across different states

## Why I built this
I built this project to practice a full workflow, from raw data to dashboard, and to get more comfortable translating business questions into analysis that people can actually use.
