# Superstore Sales Analysis with Python

## Project Overview
This project analyzes a Superstore sales dataset using Python.  
The goal is to understand sales performance, profitability, customer segments, product categories, and return behavior.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains retail business data, including:
- Order Date
- Sales
- Profit
- Segment
- Region
- State
- Category
- Sub-Category
- Product Name
- Returns

## Project Objectives
This project aims to answer the following questions:
- How do sales change over time?
- Which categories and sub-categories are the most and least profitable?
- Which customer segment performs best?
- Which states have the lowest profit?
- Do returned orders have lower profit margins?

## Key Insights
1. Sales peaked around early 2020 before stabilizing.
2. Consumer is the strongest segment in both sales and profit.
3. Tables, Supplies, and Bookcases are the weakest sub-categories by profit.
4. Tables generate high sales but still produce the largest loss.
5. Copiers, Accessories, Paper, and Phones are the most profitable sub-categories.
6. Texas has the largest loss despite having high sales.
7. Returned orders have lower average profit margins than non-returned orders.

## Recommendations
1. Review pricing and discount strategies for loss-making sub-categories.
2. Focus more on high-profit sub-categories such as Copiers and Accessories.
3. Investigate weak-performing states, especially Texas and Illinois.
4. Reduce return-related losses by reviewing products with low margins.
